1. Download pidgin source from http://www.pidgin.im/download/source/
2. Copy libpurple to a library folder (e.g. C:/Libraries/libpurple)
3. Download glib-2.39.0 from https://download.gnome.org/sources/glib/2.39/
4. Move it into the same folder (C:/Libraries/glib-2.39.0)
5. Set CMAKE_LIBRARY_PATH to your library folder (C:/Libraries)
6. Run cmake
