\subsection{\-Detailed \-Description}
\-A \hyperlink{classwhisper__library_1_1ChannelManager}{\-Channel\-Manager} is the connecting part for the covert channel and the framework. 

\-You can add and remove covert channels you want to use. \-You pass the data you want to send to this class. \-It is connected to the network via a \hyperlink{classwhisper__library_1_1SocketConnector}{\-Socket\-Connector}.
